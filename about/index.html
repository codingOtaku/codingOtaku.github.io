About



If you are wondering why I chose the name "Coding Otaku", I came up with it while searching for a name that's related to both programming and Anime.
No one bought that as a domain name, and I couldn't find anyone in social media going by that name (surprise!).

This website contains some of my projects, links to other sites, and ways to contact me.
This was never intended to be my portfolio or resume, I treat it my personal space.
